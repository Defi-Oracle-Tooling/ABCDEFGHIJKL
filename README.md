## Getting Started

1. Install dependencies:
   ```
   pnpm install
   ```
2. Run the development server:
   ```
   pnpm dev
   ```
3. Build for production:
   ```
   pnpm build
   ```
4. Run tests:
   ```
   pnpm test
   ```
5. Lint the code:
   ```
   pnpm lint
   ```
